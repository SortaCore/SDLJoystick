SDL Joystick Object - Expressions

Number of devices
Returns the number of devices currently connected. Up to 16 devices can be connected simultaneously.

Get axis value
Returns the current value of a specific axis (-32768 to 32768). Device IDs range from 0 to 15, axis IDs range from 0 to 7.

Get hat value
Returns the value of a specific POV hat/D-Pad on a device. Device IDs range from 0 to 15, hat IDs range from 0 to 3. Return values are as follows:
0 - centered
1 - up
2 - right
3 - up + right
4 - down
6 - down + right
8 - left
9 - up + left
12 - down + left

Get ball X delta
Returns the X position difference (from the previous frame) for a specific trackball on a device. Device IDs range from 0 to 15, ball IDs range from 0 to 3.

Get ball Y delta
Returns the Y position difference (from the previous frame) for a specific trackball on a device. Device IDs range from 0 to 15, ball IDs range from 0 to 3.

Get button state
Returns 1 if a specific button on a device is pressed or held down, and 0 if not. Device IDs range from 0 to 15, button IDs range from 0 to 31.

Get raw buttons (HEX)
Returns a HEX string containing bit representations of all buttons currently pressed on a device. Device IDs range from 0 to 15.

Get held button index
Returns the index of a button currently held down on a device. Up to 32 button slots can be tested for multiple buttons being pressed at the same time. Device IDs range from 0 to 15, button slots range from 0 to 31.

Get last button pressed
Returns the index of the last button that was pressed on a device. This value only updates when a new button is pressed. Device IDs range from 0 to 15.

Get last button released
Returns the index of the last button that was release on a device. This value only updates when a new button is released. Device IDs range from 0 to 15.

Get device name
Returns a string containing the device name (as reported by SDL). Device IDs range from 0 to 15.

Get device GUID
Returns a string containing the unique identifier (GUID) for a specific device. Device IDs range from 0 to 15.

Number of axes
Returns the number of axes supported by a specific device. Device IDs range from 0 to 15. The maximum number of axes is 8.

Number of buttons
Returns the number of axes buttons by a specific device. Device IDs range from 0 to 15. The maximum number of buttons is 32.

Number of hats
Returns the number of hats/D-Pads supported by a specific device. Device IDs range from 0 to 15. The maximum number of hats is 4.

Number of balls
Returns the number of trackballs supported by a specific device. Device IDs range from 0 to 15. The maximum number of balls is 4.
